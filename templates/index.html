Different Brushes Test

This is an experimental project that specifies in testing different, unique, or even custom brushes for drawing.
It is a work in progress, and will be updated frequently.

What is this?

Here we will be testing different brushes for drawing things in a web interface. To get started, simply open the app and start drawing!

How do I use this?

Simply open the app and start drawing! You can change the brush by clicking on the brush icon in the top left corner of the screen.

Can I add my own brush?

Sure! You can try importing the .cebt file into the app, which will add the brush to the list of brushes. .cebt files are generated from normal images so they'll work good as actual brushes.
